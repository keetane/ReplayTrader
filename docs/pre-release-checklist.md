# 公開前チェックリスト

最終更新日: 2026-07-06

Replay Trader を GitHub Pages などで無料公開する前に確認する項目です。法的な最終判断を代替するものではありませんが、実務上の抜け漏れ確認に利用できます。

## 1. 公開形態

- [ ] 公開先 URL が確定している。
- [ ] 公開対象は静的ビルドであり、ユーザー CSV を受け取るバックエンドがない。
- [ ] GitHub Pages の公開ブランチ、`public/`、`dist/`、サンプルアセットに実在相場 CSV が含まれていない。
- [ ] 開発者用テストデータが public repository にコミットされていない。
- [ ] 検証環境に実在データを置く場合、公開環境と分離し、必要に応じてパスワード保護している。
- [ ] スクリーンショット、動画、README、Issue、PR、デモページに、権利関係が不明な実在データが大きく写っていない。

## 2. 金融機能・表示

- [ ] 実注文機能がない。
- [ ] 証券会社、FX 業者、暗号資産交換業者、銀行などの API に接続していない。
- [ ] 口座 ID、API キー、認証情報、証券口座情報の入力欄がない。
- [ ] 売買推奨、銘柄推奨、投資助言、シグナル配信、自動売買を行っていない。
- [ ] 「買うべき」「売るべき」「勝てる」「利益保証」などの表現が UI、README、広告文、SNS 投稿にない。
- [ ] 仮想注文、仮想約定、建玉、損益が実取引ではないことを画面または文書で説明している。
- [ ] 手数料、税金、金利、貸株料、逆日歩、スリッページ、流動性、取引規制を完全には再現しないことを説明している。

## 3. CSV とデータ権利

- [ ] ユーザー CSV はブラウザ内で処理され、運営者管理サーバーへアップロードされない。
- [ ] CSV 読み込み処理に、外部送信、エラーレポート送信、解析送信が混ざっていない。
- [ ] サンプルデータは架空データまたは再配布可能性が確認できるデータだけである。
- [ ] README と文書で、利用者が CSV の利用権限を確認する必要があることを明記している。
- [ ] エクスポート機能がある場合、出力に CSV 由来データや仮想売買履歴が含まれる可能性を説明している。
- [ ] クラウド保存、同期、共有、ランキング、投稿機能を公開版に含めていない。含める場合は、別途データ取扱いを見直している。

## 4. プライバシー

- [ ] プライバシーポリシーを公開している。
- [ ] CSV 本体、仮想注文、仮想損益を通常利用では運営者が取得しないことを説明している。
- [ ] IndexedDB、localStorage、ブラウザキャッシュなどのローカル保存について説明している。
- [ ] 問い合わせ先、運営者名、または連絡手段を公開前に記入している。
- [ ] アクセス解析を導入する場合、ツール名、取得情報、利用目的、オプトアウト方法を追記している。
- [ ] 広告を導入する場合、広告事業者名、Cookie、広告識別子、パーソナライズ広告、オプトアウト方法を追記している。
- [ ] EEA、UK、スイス向けに広告配信する場合、広告事業者が求める同意管理要件を確認している。

## 5. 利用規約・免責

- [ ] 免責事項を公開している。
- [ ] 利用規約を公開している。
- [ ] データ取扱方針を公開している。
- [ ] 本アプリが投資助言、売買推奨、実注文を提供しないことを明記している。
- [ ] 利用者が読み込む CSV の権利確認責任を明記している。
- [ ] 無料提供であり、継続提供や完全性を保証しないことを明記している。
- [ ] 広告収入の可能性を踏まえた記載になっている。

## 6. 商標・外部サービス

- [ ] アプリ名、README、公開ページ、メタタグに `TradingView` を含めていない。
- [ ] 「TradingView クローン」「TradingView 互換」など、混同を招く表現を避けている。
- [ ] TradingView のデータ、API、スクリーンショット、チャート画像、Pine Script、ブランド素材を無断利用していない。
- [ ] `lightweight-charts` など OSS ライブラリのライセンス表示を確認している。
- [ ] 外部リンク、広告、問い合わせフォームなどの第三者サービスの規約を確認している。

## 7. GitHub Pages 配信

- [ ] 公開ビルドに `.env`、秘密鍵、API キー、個人ファイル、テスト CSV が含まれていない。
- [ ] 404 ページ、ベースパス、アセットパスが GitHub Pages で動作する。
- [ ] README から公開用文書へリンクしている。
- [ ] 公開ページからも免責事項、利用規約、プライバシーポリシー、データ取扱方針へアクセスできる。
- [ ] 公開前に別ブラウザまたはシークレットウィンドウで、CSV がアップロードされないことを開発者ツールの Network タブで確認している。

## 8. 広告導入時の追加確認

- [ ] 広告導入前にプライバシーポリシーを更新している。
- [ ] 広告事業者の必須表示、禁止コンテンツ、誤クリック誘導禁止、配置ルールを確認している。
- [ ] パーソナライズ広告を使うか、非パーソナライズ広告にするかを決めている。
- [ ] Cookie バナーや同意管理が必要な対象地域を確認している。
- [ ] 投資・金融関連コンテンツに対する広告ポリシーを確認している。

## 9. 公開後の運用

- [ ] 問い合わせや削除依頼に対応できる連絡手段がある。
- [ ] 規約、プライバシーポリシー、データ取扱方針の更新日を管理している。
- [ ] 実注文、助言、シグナル、クラウド保存などの新機能を追加する前に、文書とリスク評価を見直す運用にしている。
- [ ] 重大な仕様変更、広告導入、外部送信追加があった場合、README と公開ページの説明を更新する。
