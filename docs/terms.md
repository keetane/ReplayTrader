# 利用規約

最終更新日: 2026-07-06

この利用規約（以下「本規約」といいます）は、Replay Trader（以下「本アプリ」といいます）の利用条件を定めるものです。本アプリを利用する場合、利用者は本規約に同意したものとみなされます。

## 1. 本アプリの内容

本アプリは、利用者が自分で用意した CSV データをブラウザ内で読み込み、過去データの閲覧、リプレイ、仮想売買の練習を行うためのトレーニング用ツールです。

本アプリは、実注文、投資助言、売買推奨、銘柄推奨、シグナル配信、自動売買、証券 API 連携、資産運用の受託、注文の媒介・取次・代理を提供しません。

利用者は、読み込む CSV データについて、提供元の利用規約、著作権、データライセンス、商用利用条件、再配布制限、契約上の制限を確認し、必要な権利または許諾を有する範囲で利用してください。

利用者は、本アプリ上の表示や仮想売買結果を、実際の取引成果や将来の投資成果を保証するものとして扱わないものとします。

## 3. CSV データとローカル保存

本アプリは、CSV データを利用者のブラウザ内で処理する設計です。通常の利用において、CSV ファイルの内容を本アプリの運営者が管理するサーバーへアップロードすることは想定していません。

本アプリは、利用者の選択やブラウザの機能により、リプレイセッション、仮想注文、約定履歴、設定、CSV に由来するデータを IndexedDB、localStorage、その他のブラウザ内ストレージに保存する場合があります。

利用者は、共用端末、職場端末、学校端末などで本アプリを利用する場合、ローカル保存されたデータが端末利用者や管理者から閲覧される可能性に注意してください。
