# データ取扱方針

最終更新日: 2026-07-06

このデータ取扱方針は、Replay Trader（以下「本アプリ」といいます）で扱う CSV データ、リプレイデータ、仮想売買データの取扱いを説明するものです。

## 1. 対象データ

本方針の対象は、主に次のデータです。

- 利用者が選択した CSV ファイル
- CSV から読み取られる日時、始値、高値、安値、終値、出来高などのデータ
- CSV ファイル名から推定される銘柄名または識別子
- リプレイ開始位置、再生速度、表示足、チャート設定
- 仮想注文、仮想約定、仮想建玉、評価損益、確定損益、履歴
- IndexedDB、localStorage、エクスポートファイルなどに保存されるアプリ内データ

## 2. ブラウザ内処理

本アプリは、CSV データを利用者のブラウザ内で処理する設計です。CSV ファイルの選択、読み込み、パース、検証、チャート表示、リプレイ、仮想売買の計算は、原則として利用者の端末上で行われます。

通常の利用において、CSV ファイルの内容を本アプリの運営者が管理するサーバーへアップロードすることは想定していません。GitHub Pages で公開する場合も、本アプリは静的ファイルとして配信され、CSV の処理はブラウザ側で完結します。

## 3. ローカル保存

本アプリは、利便性のために、次の情報をブラウザ内ストレージに保存する場合があります。

- 読み込み済みデータに由来する情報
- リプレイセッション
- 仮想注文、約定履歴、建玉、損益
- 画面設定、テーマ、リプレイ設定

保存先には IndexedDB、localStorage、sessionStorage、ブラウザキャッシュなどが含まれる場合があります。これらは原則として利用者の端末内に保存され、本アプリの運営者が直接閲覧することは想定していません。

利用者は、ブラウザのサイトデータ削除、アプリ内削除機能、シークレットモードの利用、端末の初期化などにより、ローカル保存データを削除または抑制できる場合があります。

## 4. アップロードしないデータ

本アプリは、通常の利用において次のデータを運営者管理サーバーへ送信しない設計です。

- CSV ファイル本体
- CSV の中身から生成される価格、出来高、時刻、銘柄データ
- 仮想注文、仮想約定、建玉、損益履歴
- リプレイセッションの詳細

将来、同期機能、クラウド保存、共有機能、問い合わせ添付、エラーレポート、アクセス解析などを追加する場合は、送信対象、利用目的、保存期間、第三者提供の有無を事前に文書化し、必要に応じて同意取得や設定変更を行います。

## 5. 利用者が確認すべきこと

利用者は、CSV データを読み込む前に、次の点を確認してください。

- データ提供元の利用規約で、保存、加工、アプリへの読み込み、商用利用、第三者表示、派生データ作成が許されているか
- データ提供元が、スクレイピング、再配布、共有、公開、広告付きサービスでの利用を制限していないか
- CSV に個人情報、口座情報、注文履歴、メモ、API キー、認証情報などが含まれていないか
- 共用端末や業務端末で利用する場合、ローカル保存やブラウザ同期によって第三者に見られる可能性がないか

本アプリは、利用者が読み込む CSV データの権利関係や契約条件を確認しません。利用者は、自分が利用できる範囲のデータだけを読み込んでください。

## 6. サンプルデータ

公開ビルドに含めるサンプルデータは、架空データまたは再配布が明確に許可されたデータだけにします。実在の相場データ、証券会社や情報ベンダーから取得したデータ、第三者の権利がある可能性のあるデータは、公開リポジトリ、`public/`、`dist/`、GitHub Pages の公開ブランチに含めない方針です。

開発者が動作確認のために実在データを利用する場合でも、公開リポジトリや公開 URL に含めず、必要に応じてパスワード保護された検証環境で扱います。

## 7. エクスポートと共有

本アプリにエクスポート機能がある場合、出力ファイルには CSV に由来する情報、仮想売買履歴、設定などが含まれる可能性があります。利用者は、エクスポートファイルを共有、公開、送信する前に、含まれるデータと権利関係を確認してください。

本アプリの運営者は、利用者が任意に共有したファイルやスクリーンショットに含まれる情報について、事前確認できません。

## 8. 広告・外部サービスとの関係

将来広告を導入する場合でも、CSV ファイル本体や仮想売買履歴を広告事業者へ送信しない設計を維持します。ただし、広告事業者は、広告表示のために Cookie、広告識別子、IP アドレス、ユーザーエージェント、ページ URL、広告表示・クリック情報などを取得する場合があります。

広告、アクセス解析、問い合わせフォーム、外部リンクなどの第三者サービスを導入する場合は、プライバシーポリシーに反映します。

## 9. 方針変更

クラウド保存、ユーザーアカウント、共有、ランキング、実データ配信、証券 API 連携、実注文、売買シグナル、自動売買、広告配信、アクセス解析を追加する場合は、本方針を見直します。

特に、CSV をサーバーへアップロードする機能、他ユーザーと共有する機能、実在相場データを配布する機能を追加する場合は、データライセンス、プライバシー、金融規制、広告規約の観点から事前確認が必要です。
