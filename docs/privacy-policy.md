# プライバシーポリシー

最終更新日: 2026-07-06

このプライバシーポリシーは、Replay Trader（以下「本アプリ」といいます）における情報の取扱い方針を説明するものです。

## 1. 基本方針

本アプリは、利用者が自分で用意した CSV データをブラウザ内で読み込み、過去データの閲覧、リプレイ、仮想売買の練習を行うためのトレーニング用ツールです。

本アプリは、通常の利用において、CSV ファイルの内容を本アプリの運営者が管理するサーバーへアップロードすることを想定していません。CSV の読み込み、パース、チャート表示、リプレイ、仮想売買、セッション保存は、利用者のブラウザ内で行われます。

## 2. 本アプリが取得しない情報

現時点の想定では、本アプリの運営者は、通常の利用において次の情報を直接取得しません。

- 利用者が読み込んだ CSV ファイルの内容
- CSV から生成された価格、出来高、銘柄、仮想注文、仮想約定、損益履歴
- 氏名、住所、電話番号、証券口座情報、銀行口座情報、本人確認書類
- 実際の注文情報、取引口座情報、資産残高

本アプリは実注文機能を持たず、証券会社、金融機関、暗号資産交換業者、FX 業者などの口座情報の入力を求めません。

## 3. ブラウザ内に保存される情報

本アプリは、利用者の操作や設定に応じて、次の情報を利用者のブラウザ内に保存する場合があります。

- 読み込み済みデータに由来する情報
- リプレイセッション
- 仮想注文、仮想約定、仮想建玉、損益履歴
- 画面表示、テーマ、リプレイ速度などの設定
- エクスポートまたはインポート用のデータ

保存先には、IndexedDB、localStorage、sessionStorage、ブラウザキャッシュなどが含まれる場合があります。これらは原則として利用者の端末内に保存されます。

ブラウザ内に保存されたデータは、利用者がブラウザのサイトデータ削除、キャッシュ削除、アプリ内削除機能、シークレットモードの終了などを行うことで削除できる場合があります。ただし、ブラウザや端末の仕様により挙動が異なることがあります。

## 4. アクセス情報

本アプリを GitHub Pages などのホスティングサービスで公開する場合、ホスティング事業者がアクセスログ、IP アドレス、ユーザーエージェント、リファラー、日時などを取得する可能性があります。

本アプリの運営者が独自にアクセス解析ツールを導入する場合は、導入するツール名、取得する情報、利用目的、オプトアウト方法を本ポリシーに追記します。

## 5. Cookie と広告

現時点で広告を表示しない場合、本アプリの運営者は広告配信用 Cookie を自ら設定しません。

将来、Google AdSense などの広告サービスを導入する場合、広告事業者が Cookie、広告識別子、端末情報、アクセス情報、広告表示・クリック情報などを取得または利用する可能性があります。その場合は、広告事業者名、利用目的、第三者提供、パーソナライズ広告の有無、オプトアウト方法、同意管理の有無を本ポリシーに追記します。

EEA、UK、スイスなど、広告 Cookie やパーソナライズ広告について追加の同意管理が求められる地域に配信する場合は、広告事業者の要件に従って同意管理の導入を検討します。

## 6. 利用目的

本アプリの運営者が問い合わせなどを通じて利用者から情報を受け取る場合、その情報は次の目的で利用します。

- 問い合わせへの回答
- 不具合、権利侵害、規約違反への対応
- 本アプリの改善、保守、公開運用
- 法令、規約、ポリシーに基づく対応

## 7. 第三者提供

本アプリの運営者は、法令に基づく場合、利用者の同意がある場合、問い合わせ対応やホスティング、広告、アクセス解析などの外部サービス利用に必要な範囲を除き、取得した利用者情報を第三者に提供しません。

ただし、本アプリは GitHub Pages、広告事業者、アクセス解析サービス、問い合わせフォーム、外部リンク先などの第三者サービスと併用される場合があります。これらのサービスには、それぞれのプライバシーポリシーや利用規約が適用されます。

## 8. 安全管理

本アプリの運営者は、取得した情報がある場合、その内容やリスクに応じて合理的な安全管理に努めます。

利用者は、共用端末での利用、ブラウザ拡張機能、端末のマルウェア、画面共有、スクリーンショット、バックアップ、同期機能などにより、ブラウザ内のデータが第三者に見られる可能性に注意してください。

## 9. 開示・訂正・削除等

本アプリの運営者が利用者情報を保有している場合、利用者本人からの開示、訂正、削除、利用停止などの依頼について、法令や本人確認の必要性を踏まえて合理的な範囲で対応します。

ブラウザ内に保存されたデータは、原則として利用者の端末内にあるため、利用者自身で削除してください。
